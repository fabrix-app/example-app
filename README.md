An example Fabrix Application.

## 1. Install
Clone or fork this repository and then run:
```sh
$ npm install
```

## 2. Build
The src must be built.
```sh
$ npm run build
```

## 3. Start
Then Start the server.
```sh
$ node dist/server
```
